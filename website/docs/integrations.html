Integrations directory

104 sources and 59 sinks that work today, each covered by tests. Search the catalogue or filter by category and direction. Sources read in, destinations write out.

104 sources

Read from files, databases, warehouses, object storage, streams, NoSQL stores, vector DBs, and SaaS APIs.

128 transforms

Shape, join, enrich, and validate data between source and sink. See the component reference.

59 destinations

Write to files, databases, warehouses, object storage, vector DBs, messaging, and mailboxes.

Roadmap. Roadmap connectors (e.g. IBM DB2, Pulsar sink, Event Hubs, multi-shard Kinesis, Chroma, LanceDB) are reserved in the palette and shipping engine-by-engine. All upserts support optional CDC delete propagation. For the transforms and quality nodes that run between source and sink, see the component reference.
